## Project Details  

### Customer Segmentation & Retention Analysis  

A machine learning project that groups customers into meaningful segments using K-Means Clustering.  

Includes a Streamlit web application for real-time predictions.

**Key Value:**  
- Identifies high-value customers  
- Detects low-engagement users  
- Supports targeted marketing strategies  

---

### Time Series Forecasting  

A forecasting project that analyzes time-dependent data to predict future trends.  

Focuses on identifying patterns such as trend and seasonality and building a predictive model.

**Key Value:**  
- Enables data-driven future predictions  
- Useful for demand, sales, or trend forecasting  
- Demonstrates time series modeling capability
